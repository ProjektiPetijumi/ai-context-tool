# AI Context Tool — Kursa projekts

Projekts izstrādāts kā daļa no datu analīzes kursa ar MySQL, Python un Power BI.

## Uzdevums A — Power BI atskaite
- `ai_context_tool.pbix` — Power BI dashboard ar 2 lapām
- Lapa 1: KPI kartītes, stabiņu grafiki, gredzena diagramma, tabula
- Lapa 2: 4 Python vizuāļi (histogramma, boxplot, scatter, stabiņu grafiks)
- `Savienojumu kpsavilkums.jpg` — LLM ieteiktie savienojumi
- `BIZNESA ANALĪZES PLĀNS.jpg` — LLM izstrādāts analīzes plāns

## Uzdevums B — Python AI rīks
- `main.py` — datubāzes struktūras izgūšana un LLM analīze
- `main2.py` — HTML atskaites ģenerēšana ar vizualizācijām
- `atskaite.html` — ģenerētā HTML atskaite ar grafiki un AI aprakstiem
- `requirements.txt` — nepieciešamās bibliotēkas

## Izmantotās tehnoloģijas
- Python, MySQL, Gemini API, matplotlib, pandas, BeautifulSoup
- Power BI Desktop (ar Python vizuāļiem)
