# AI Context Tool

Rīks, kas izmanto LLM lai analizētu MySQL datubāzi un ģenerētu HTML atskaiti.

## Faili
- `main.py` — datubāzes konteksta izgūšana
- `main2.py` — HTML atskaites ģenerēšana ar vizualizācijām
- `atskaite.html` — ģenerētā atskaite
- `ai_context_tool.pbix` — Power BI atskaite

## Izmantotās tehnoloģijas
- Python, MySQL, Groq/Gemini API, matplotlib, pandas
